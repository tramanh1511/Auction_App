## Local Run

Clone the project

## Start client

Navigate to the project directory

```bash
   cd MagicPost/client
```

Install dependencies

```bash
   npm install
```

Start client

```bash
   npm run dev
```

## Start server

Navigate to the project directory

```bash
   cd MagicPost/server
```

Install dependencies

```bash
   npm install
```

Start server

```bash
   npm start
```
